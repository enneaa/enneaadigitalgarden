# 202303200752 文章收藏
#思  
网上收藏的文章要不要存放在笔记软件中，这是一个问题，也有点纠结。
这些文章存在笔记中，而笔记是一种比较重的载体，放太多总是一种负担。而且当文章都存在笔记中，实际上笔记软件就成了另外一种用途，也就是阅读和存档的地方。
所以，文章就用稍后读软件来管理，更简单的甚至可以用书签来管理。
